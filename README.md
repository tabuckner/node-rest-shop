# Node Rest Shop

Simple REST API example.

## Description

Mock API using JWT auth, MongoDB Atlass, and a small collection of milddleware. Attempts to follow an M<V>C approach.

## Getting Started

### Dependencies

- Node
- MongoDB Atlas account/project.

### Installing

- npm i

### Executing program

Run nodemon

```
npm start
```
